Privacy Policy for Finance Notes

Last Updated: [Date]

This Privacy Policy describes how Finance Notes ("we," "our," or "us") collects, uses, and protects your information when you use our mobile application (the "App").

1. Information We Collect

1.1 Personal Information

When you register and use our App, we collect the following information:

  • Name: Your full name for account identification
  • Email Address: Required for account creation and communication
  • Phone Number: Used for authentication and transaction management
  • Aadhaar Number: Collected for identity verification and transaction purposes (highly sensitive data)
  • Profile Picture: If you sign in using Google, we may collect your Google profile picture

1.2 Financial Information

  • Transaction amounts and details
  • Transaction status (pending, closed, cancelled)
  • Interest calculations
  • Customer names associated with transactions

1.3 Documents

  • Transaction-related documents (receipts, agreements, etc.) that you upload
  • Document metadata (filename, size, type)

1.4 Communication Data

  • Chat messages sent through the App
  • Message timestamps and status

1.5 Authentication Data

  • Google Sign-In credentials (if you choose Google authentication)
  • Encrypted password (if using email/password authentication)
  • Authentication tokens (stored securely on your device)

1.6 Technical Information

  • Device information
  • App usage data
  • Log files and error reports

2. How We Use Your Information

We use the collected information for the following purposes:

2.1 Service Provision

  • To create and manage your account
  • To process and manage financial transactions
  • To facilitate communication between users (chat functionality)
  • To verify user identity using Aadhaar verification services

2.2 Transaction Management

  • To record and track financial transactions
  • To calculate and apply interest
  • To manage transaction documents
  • To enable transaction closure and OTP verification

2.3 Communication

  • To enable chat functionality between transaction parties
  • To send OTP (One-Time Password) for transaction verification
  • To send important service notifications

2.4 Security and Compliance

  • To verify user identity
  • To prevent fraud and unauthorized access
  • To comply with legal obligations
  • To ensure data security and integrity

3. Third-Party Services

Our App uses the following third-party services that may collect information:

3.1 Google Sign-In

  • We use Google Sign-In for authentication
  • Google may collect information according to their Privacy Policy: https://policies.google.com/privacy

3.2 Aadhaar Verification Services

  • We use third-party Aadhaar verification services (QuickeKYC)

3.3 Data Storage

  • User data is stored on secure cloud servers
  • All data transmission is encrypted using industry-standard protocols

4. Data Storage and Security

4.1 Data Storage

  • Your personal information is stored in secure databases
  • Passwords are encrypted using industry-standard hashing algorithms
  • Aadhaar numbers are stored securely and handled with utmost care

4.2 Security Measures

  • We implement appropriate technical and organizational security measures
  • Data transmission is encrypted using SSL/TLS protocols
  • Access to personal data is restricted to authorized personnel only
  • Regular security audits and updates are performed

4.3 Data Retention

  • We retain your personal information as long as your account is active

  • You can access and update your personal information through the App settings

5.2 Data Deletion

5.3 Data Portability

5.4 Withdrawal of Consent

6. Children's Privacy

Our App is not intended for children under the age of 13. We do not knowingly collect personal information from children under 13. If you believe we have collected information from a child under 13, please contact us immediately.

7. Aadhaar Data Protection

  • Comply with the Aadhaar Act, 2016 and related regulations
  • Use Aadhaar data only for identity verification purposes
  • Implement additional security measures for Aadhaar data
  • Never share Aadhaar numbers with unauthorized parties
  • Allow Aadhaar data usage only through registered Authentication User Agencies (AUAs)
